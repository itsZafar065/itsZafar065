## 🛠️ GitHub Actions Setup: Automated Contribution Snake
To enable the contribution snake animation seen above, you need to set up a GitHub Actions workflow in your profile repository. Follow these steps:
1. Create a directory named `.github/workflows` in your repository.
2. Inside that directory, create a file named `generate-snake.yml`.
3. Paste the following YAML configuration inside `generate-snake.yml`:
```yaml
name: Generate Snake
on:
  schedule: # Execute every 12 hours
    - cron: "0 */12 * * *"
  workflow_dispatch: # Allows manual trigger
  push:
    branches:
      - main
jobs:
  generate:
    runs-on: ubuntu-latest
    timeout-minutes: 10
    steps:
      # Generates a snake game from a github user's contributions graph, output an SVG animation
      - name: Generate github-contribution-grid-snake.svg
        uses: Platane/snk/svg-only@v3
        with:
          github_user_name: itsZafar065
          outputs: |
            dist/github-contribution-grid-snake.svg
            dist/github-contribution-grid-snake-dark.svg?palette=github-dark
      # Push the content of <build_dir> to a branch
      # The content will be available at https://raw.githubusercontent.com/<username>/<repo>/<branch>/<path>
      - name: Push github-contribution-grid-snake.svg to Output Branch
        uses: crazy-max/ghaction-github-pages@v3.1.0
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```
4. Go to your repository settings on GitHub: **Settings > Actions > General > Workflow permissions** and select **"Read and write permissions"**, then click **Save**.
5. Run the workflow manually from the **Actions** tab of your repository to generate your first snake!
